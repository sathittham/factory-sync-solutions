**Release Summary:**
Backoffice GA4 analytics (CR-006 + CR-007): a new backend `services/analytics`
GA4 Data API proxy (seven endpoints, 15m TTL cache, stale-while-error) and a
dedicated `/analytics` page in `web-backoffice` with a sidebar menu item and
per-surface site tabs (All / Official website / Web app via GA4 `hostName`
filtering). Also reworks the `web-app` dashboard into KPI stat cards, clears
repo-wide Biome lint debt (a11y), and wires GA4 runtime config into the Cloud
Run deploy workflows via Secret Manager.
