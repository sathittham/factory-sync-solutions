**Release Summary:**
Completes the TanStack Query rollout in `web-app` (CR-003): server-state fetching
moves off hand-rolled `fetch` + `useState` + `useEffect` and off Redux mirrors onto
`@tanstack/react-query`. The `result` Redux slice is retired, the `quiz` slice is
trimmed to client state, and profile writes are centralized behind a single
`useUpdateProfileMutation`. Internal refactor — no user-facing behavior change.
